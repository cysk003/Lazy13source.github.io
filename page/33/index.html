关于养猪行业我再谈

2019-03-17 #caoz密圈
禺 : 关于养猪行业我再谈几个问题：1:2月能繁母猪存栏数据出来了，环比继续大幅下降5%，同比下滑接近20%，3月的话能繁下滑可能要扩大到25%。这里我得说一下，外行看热闹，内行看门道，2月过节，屠宰企业一般都要休息十天半个月不收猪，所以从下图的监测样本母猪屠宰数量也能看到，每年过节屠宰淘汰母猪数量最少，但今年2月能繁母猪存栏又下滑5%，减少130-140万头猪，那这里只能是由疫情影响以及初配母猪的大量减少导致，这个影响因素会很长时间存在。2:关于猪价，猪价又开始涨了，目前看屠企压价失败了，河南山东全部都是16元多了，猪价在最淡季节都压不住都能涨到这么高，那到了今年4季度需求旺季，同时叠加生猪出栏的大幅缩减（2018年12月开始能繁母猪的存栏开始大幅下降，影响10个月以后的生猪出栏），猪价创历史新高成为必然。可能不用等到四季度，很多人预计6，7月份没准就要奔新高去了，这波猪周期已经启动，未来价格要涨到很多人难以置信，这是供求决定。说影响cpi的看我之前文章。3；关于进口，前两天一个新闻，说从美国进口了23846吨猪肉，23846吨=23846000公斤 ，一只出栏猪的标准重量是110公斤，就相当于这次采购大概20万头猪，而未来中国生猪缺口大概多大，按30%算估计得有2亿缺口，这20万头猪相当于缺口的0.1%。2019-03-17点赞数:156

我今天谈一个重要问

2019-03-16 #caoz密圈
禺 : 我今天谈一个重要问题，关于成长股爆发性高增长的两种模式，我之前文章写过我的选股的核心逻辑是企业业绩高爆发成长，周期股业绩爆发力强这个大家好理解，那成长股的业绩爆发力怎么体现的，一般来说有两种方式，第一种方式公司突然出现了一款特别牛的产品带动公司利润出现爆发性增长。另一种方式就是某个公司出现了一系列可能成为爆款的新产品，比如当年中恒血栓通就这一个产品让公司飞速成长好几年，再比如2012-2013年长安福特，一下投入12款新车，同时有多个爆款。那可能有人会问怎么才能知道公司产品会不会成为爆款，那你要是关注游戏行业，taptap你得关注吧，游戏上市你得自己玩玩吧，游戏流水你得留意吧。你什么都不关注就想坐享其成投资可能成功吗？只有短期业绩高爆发，你才有可能获得高额投资利润，才能回避一些不确定的风险。黑马通常都是诞生于周期股和成长股中的业绩高爆发股，困境反转股，以及重组股，重组股一般得有深厚人脉关系否则一般人做不了这种股票。好了我在这里抛砖引玉，其他大家自己研究。2019-03-16点赞数:146评论区：宁酷到底 : 先👍在读[呲牙] 让子弹再飞一会儿 : 先赏在读 大宝天天见 : 谢谢梁老师 禺 : 我和大家说一下，在这里我不可能把所有的我的投资系统都告诉大家，我就简单说说框架，其他只能你们自己去研究了，比如怎么去选择股票，分析行业，不同企业怎么去估值，怎么去跟踪企业经营，这都要靠你们自己去悟了 谭兆柏 : 看得懂一个细分行业就需要投入对应的功夫了 芒果 : 看来老师要开辟新战场了，期待 ling : 梁老师说的深有感触[强] King金哥 : 这段话能理解并运用就很不错了[呲牙]
