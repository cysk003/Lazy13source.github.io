# 中国电影缺的并不只是韩国电影的所谓“敢拍”

[**返回列表页**](/gzh/记忆承载3)

公众号【懒人搜索】备份，仅供查阅学习

这个春节档期，很多读者跟我抱怨说电影很烂。

说不理解为啥拍成那样，难道是为了解决就业么？

你比如那个封神2，片尾3个彩蛋前长得无边无际的工作人员列表，目测几千人。

不是说这几千人的工作不到位，非常到位，细节可以看到都很用心，但是掩盖不了剧情上的混乱。  

包括射雕。

画面，音乐，道具，镜头这些都没问题，但是剧本，就是一个综艺节目的水平。  

一个企业年会上员工们随手编排的剧本水平，你就算花几个亿，再大的制作下，也掩盖不了拉垮的剧情，混乱的节奏。

........

关于你们说的这个问题，多年以来，已经被讨论过无数次了。

有人讲，我们的电影不好是因为没有韩国人那么敢拍。

韩国电影的尺度是很大的，那个审核标准近乎于不存在，什么都敢演，什么都敢拍，什么台词都敢说。  

但我个人认为，问题不在这里，问题在于资源的分配。  

你审核无非决定了题材，总共有100个题材，砍掉90个，不还有10个么？  

这10个，你有没有拍好？

俗话说，不允许批评，至少允许表扬吧？那么表扬，有没有拍好？  

不也没拍好么？

那说明什么？说明一本电影的资金分配出了问题。  

人家韩国人拍电影，编剧占的比例是相当大的，他们拍一本电影，会花很多时间，精力，人力物力去做考证。  

这个是重点。  

就像我们87年拍摄红楼梦，第一件事，是让所有演员住一起，读红楼梦，探讨红楼梦，研究红楼梦，像红楼梦里面的人一样饮食起居。

现在的剧组呢？  

你要拍封神，大家读不读封神嘛，你要拍射雕，大家读不读射雕嘛。

如果都不读，那拍的是封神么？是射雕么？  

不，那是披着封神外衣，披着射雕外衣的流星花园，那是杉菜和F4。

这才是国产电影最大的问题，资金分配问题。

我们的资金是根据什么分配的？是根据流量的号召力来分配的。  

哪个明星在粉丝里有号召力，他就会拿走9成的预算，其余的1成要给道具，化妆，场景，等等。  

编剧有地位么？没有的，电影编剧的地位只怕跟片场的后勤，跟杂务，跟司机一样。  

电影的本质是什么？  

是讲故事。

如果剧本故事本身是一坨，那么再精美的包装下，不也是精美的一坨么？  

精美的一坨，让流量明星去号召，那不还是号召之后的精美的一坨么？  

这样的电影，用人去拍，真心不如用AI去拍。  

随着国内的AI开源在全球的推广，以后驯化一个属于自己的AI原型是很便宜的，也许几百万就够了。  

那拍电影还需要动辄投入几个亿么？  

你搭一个AI电影梦工厂，也许都只要几百万，从此它就可以给你不停的输出电影。  

你想多宏大就有多宏大，不比你用真人拍摄便宜得多？
