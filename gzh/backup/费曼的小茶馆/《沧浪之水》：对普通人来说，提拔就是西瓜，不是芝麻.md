# 《沧浪之水》：对普通人来说，提拔就是西瓜，不是芝麻

[**返回列表页**](/gzh/费曼的小茶馆)

小懒自动备份，仅供查阅学习

今晚闲着，随手翻了一下《沧浪之水》。

小说中有一个桥段：

主人公池大为探望生病的单位司机。

司机颇为感动。

他观察到池大为看不惯人情世故，不注重为人做事的小节。

于是，偷偷告诉主人公池大为：对咱普通人来说，提拔就是西瓜，不是芝麻，你把提拔看成芝麻，看不上这点东西，迟早要吃亏。  

看到这个桥段，我很感慨，这司机的话很真实很诚恳。  

几十年前，在机关单位，干部提拔是很艰难的。  

想提拔成为一个中层干部，比现在难得多。  

绝大部分的提拔，都是藏着掖着，只有小圈子才知情。  

你作为普通人，压根就不知道自己有没有机会提拔。

所以生活在那个年代，干部都是夹着尾巴做人的。

现在你看那些退休老干，为啥看起来憨憨厚厚、生性谨慎，对年轻人毕恭毕敬。  

不是他们年老，也不是他们生活的时代闭塞。  

而是因为他们上班的年代，做人就是要恭恭敬敬。

你以为他们憨憨，其实他们有丰富的斗争经验。

现在几十年过去了。  

机关职场的环境大为不同。  

什么喝茶看报，早就没有了。

上班终日是文山会海、绩效指标。  

干活连轴转，累得头晕眼花。

过去当领导的好处，比如，配备公车和司机，吃饭签单，现在都没有了。

所以，现在经常看到很多年轻干部喊躺平。

只要躺平就岁月静好了。  

至于什么人情世故、八股文章，不用去操心。  

提不提拔，就更无所谓。

这就又回到了一个问题。  

在当下的机关单位，提拔究竟是西瓜，还是芝麻？  

究竟要不要看得上提拔这件事。  

我今天只说现象。

比如，

我朋友的一个经济口单位，比绝大多数单位都要忙。

单位腾出一个中层领导职数。

这位置就是要天天写材料，写到你秃头的那种。  

就这么一个需要你打鸡血的岗位。

想争取上位的人就不下十个。

从这种竞争势头看，我压根就感受不到躺平的气息。  

又比如，  

现在机关单位，人与人之间还是很势利的。

别人看你，先看你的级别。

你说你再有才华，人品再好。  

但别人不看这个。

大家玩，也是按照级别来分圈层的。  

不是一个级别层次的，就是玩不到一块儿。

人是社交动物，特别需要社交的尊重感。

争取提拔就是能让自己贴上更好的身份标签。  

再比如，

在机关单位有一个收入待遇比较的现象。

我级别比你高，我收入一个月比你多一千几百。  

我的满足感就出来了。

如果我退休的时候，职务职级比你高，退休待遇比你好，那我这几十年的奋斗就没有白费了。

你看，机关单位，别人看你就是看级别；同事之间比较，就是比较收入。  

哪怕当年跟你一起进入机关单位的朋友。  

当年大家很年轻，都觉得，按级别看人，太俗气太市侩。

但现在十几年过去了。

你才发现，身边的朋友，真的有闰土变老的感觉。

大家都变得市侩了，可能也包括我自己。  

明明是可以看开一点的东西，就是看不开。

别人都提拔晋升，为啥我就不提拔晋升？  

所以，今晚读《沧浪之水》，我发现，人性依然没变。  

机关单位还是那个机关单位。

提拔依然是西瓜，不是芝麻。
