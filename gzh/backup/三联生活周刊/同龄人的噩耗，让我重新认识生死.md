# 同龄人的噩耗，让我重新认识生死

文｜读者：章鱼

如果初中那会儿有谁问我怕不怕死，为了面子，我铁定说不怕。但大学后，我到处给别人说自己怕死，那些危险项目千万别算上我。
**这回，我不仅是怕死那么简单，我是真切感受到了对死亡的恐惧。**

上个月我正在看“00后工作第二年，带爷爷穷游北京”的视频时，手机弹出了青青的消息，我沉浸在自己三十多岁了还窝囊的现状里，无暇顾及。

又砰砰砰的轰炸，我点开了：“宇哥肝癌晚期已经回老家休养了，是时候告别了。”“那个事业有成，月薪3w的宇哥？”“对！”

 **我没有像电视剧里演的那样悲伤，更多是恐慌，强烈的代入感，让我没有继续往下问。**

宇哥是我在广州合租时认识的。刚毕业的我，口袋空空又不愿向父母开口，装下满腔的豪情壮志南下投奔闺蜜。屋内三间房，我和闺蜜同住一间，还有一间是一对情侣，宇哥单独一间。

大家都是同龄人，屋内的人都是同学的同学，朋友的朋友，所以毕业后的第一次合租，大家关系都很融洽，好似还在大学宿舍一样。

唯一的不同就是宇哥，虽与我们同龄，但早两年工作实习，又干着吃香的工程师，会给我们很多职场建议。难得的一次聚会，宇哥问：“你们这工作怎么样，工资高吗？”“转正好多了，五千。”“啊？五千多要怎么活啊，我月薪上万都勉勉强强。”这样的理工式大直男的关心后，桌上一阵沉默，大家整顿饭都味同嚼蜡。

![](https://mmbiz.qpic.cn/mmbiz_jpg/VkpaUkchBmVicJpEIYIZTw2BGd1SJDw2GgSUmzQl6uHFuIdAXKpZ3BwmuAXib2T1kUgLu7KTO9RGISwFUbTOYFNA/640?wx_fmt=other&tp;=webp&wxfrom;=5&wx;_lazy=1&wx;_co=1)《亲爱的自己》剧照

宇哥工资虽高，但996是必然的。他晚归晚出，项目跑起来后甚至不回家，一个月能碰到他的次数也就五六次。

同一屋檐下，作息完全不同。他常常洗了衣服就出门，等他回来晾衣服，基本都是后半夜了。出于歉意，他曾提出自己买一台洗衣机，最后却因空间限制而作罢。

 **他谦逊和气的态度，让我们也接受了他直来直往的关心。对他，我们更多的是羡慕。**

996给他带来的富裕，让他在面对尴尬的情况时，可以大方潇洒地用金钱来化解。同时在生活上他也格外注重自我呵护，蛋白粉一箱箱地买，外卖也总不忘加上鲫鱼苦瓜、黄芪乌鸡等一类养生汤。这些豪气的花销，我们几个同龄人望尘莫及。都是农村出来，同是靠着自己闯荡大城市，他却住着大房间，无负担地点着外卖，吃穿用度都与我们拉开了一大截。

 **当时的我只看到了他的优渥，却未曾洞察他用健康为代价换来财富背后，实际也以一种近乎弥补的方式，竭力维持自己的健康。**

两年后因为职业发展，大家都离开广州各奔前程，联系越来越少。听闺蜜说宇哥应聘到了某头部科技公司，转战深圳，薪资飙升到每月3w上下。

后来几经辗转闺蜜又回到了原来的小区租房，我们在电话中感叹了一番时光不再，工资没涨物价飞涨。最后总结，如果是宇哥回去，肯定能全屋租下不带犹豫的。自嘲中我俩发出了窝囊废般的哈哈大笑。

**之后在同龄人的鞭策下，我逐渐从愤世嫉俗的情绪中走出，学会接受自己的平庸，并正视现实的残酷。我终于把那份对旷野的向往藏心底，转而脚踏实地地走在既定轨道上。自我安慰道，30岁觉醒，其实还不算太晚。**

![](https://mmbiz.qpic.cn/mmbiz_png/VkpaUkchBmUjoBaBRibVfyona4ZuAwTJnHWVdB19C0WICC46pgQ6ssRdr1W2qGvKegHLXxx9x4m2oYUp9ufWPdA/640?wx_fmt=other&tp;=webp&wxfrom;=5&wx;_lazy=1&wx;_co=1)《月刊家》剧照

恰在此时，短视频app也捕捉到了我的转变，推送内容帅哥美女颜值主播转向了聚焦30+年龄段，励志女性白首起家的赛道。虽然这些成功略带些颜值身材的优势，却也给了我不小的鼓舞。可奈何我长相平平，想靠颜值这条路算是断送了，靠男人更是不敢想。而心里的傲气也不允许摆烂躺平，于是我选择了最接地气的办法往去努力——用时间和健康为代价，换钱。

在很长一段时间，面对身体那些半大不小的毛病，我都直接忽视，即使是熬夜赶私单，我也能咬牙坚持下来。这样的生活方式无疑是让身体造成了巨大的消耗。为了维持白天的精神状态，我每天都用冰美式续命工作；下班后，我则直奔健身房，疯狂撸铁，试图弥补回一些身体的亏空。

这种强行灌鸡汤的方式，我副业收入终于稍微有了些起色。可还不等带父母去看天安门升国旗，他们先一步在视频中发现了我的变化。他们言语中满是关切，每天也更加频繁地发送关于“夏日养生”、“年轻人的工作病”等健康推文给我。我全当他们是关心则乱，电话里敷衍着会注意身体的，但从未打开过哪些文章。

![](https://mmbiz.qpic.cn/mmbiz_jpg/VkpaUkchBmWFp10wt9ycb8sI3t7gTygPcdYD2MKO2ejL5z8cNAY5Gyxgf1kUDKj23spDVvzcDH7vZNP0Ok29nQ/640?wx_fmt=webp&from;=appmsg)《我在他乡挺好的》剧照

在试图复刻朋友的成功足迹的路上，才取得些许甜头，他却突然传来噩耗。我黯然翻看着宇哥早已停留在了去年的朋友圈，看他昔日的事业高升、看他农村建的小豪宅、看他带父母到四处见的世面，我心中五味杂陈的。他领先了同龄人一大截，可这一切就要戛然而止了。

与长辈的离世不同，同龄人的骤然离世更让我们难以接受。事实上，我们从未被系统性地教育过如何面对死亡，如何面对疾病。父母也是对此讳莫如深，往往多谈几句，都会换以“呸呸呸”三个字。他们对健康长寿最质朴的方式就是，逢年过节进庙磕头。无论是佛祖、关二爷、还是土地公，只要遇上就奉上三炷香，台词全国统一的“菩萨保佑，今年我们全家健康长寿无病无灾财源广进，阿弥陀佛”。如果能遇到地阎王庙，那香火一定能把天庭众神都给熏了。

 **从父母这些举动中我也知道，他们的教育中，对健康生活、疾病、死亡也是一片空白的。只有用这种原始的力量去对抗生命的无常。**

而我游历人间30年里，生死的经历都是来自村里亲人、族人的仙逝，其中大部分都是有准备的。我潜意识里早已认定生死是自然规律，人老了，器官就会出问题，时间到了，人才会走。即使是读过史铁生《我与地坛》，老师们也只是强调他面对疾病时的坚韧，而对于疾病为何到来、如何面对等更深层次的问题，好像所有人都沉默了，都默认厄运不会降临到年轻的肉体上。

其实我又何尝不是带着鸵鸟思维，赖着父母还挡在身前，不愿也不敢直面死亡的问题。

![](https://mmbiz.qpic.cn/mmbiz_jpg/VkpaUkchBmWFp10wt9ycb8sI3t7gTygPHuw6Qoibb34ZLJnUy238FiaJnLAia5GLMTJve3HDmjskKKZa6n3kKgvHg/640?wx_fmt=webp&from;=appmsg)《我准时下班》剧照

而比起学校和家庭的含蓄内敛，短视频app用其强大的算法，以一种近似于残酷的方式，将死亡和疾病的真相强势推送到我面前。那些“17岁糖尿病怎么度过一天”“30岁化疗的日常”的震撼标题，伴随着真实而揪心的画面，暴力地推开了我面前那道与死亡的屏障。

宇哥的噩耗，如同一记重锤，彻底敲醒了我。我开始正视那些博主用生命来书写的教诲。我目睹他们剃去头发，以亲身经历告诫我们要注重生活习惯，警惕那些日常中不易察觉的健康威胁，然后看着他们一日日面颊凹陷、逝去血色，最后视频永远定格在了某一天。

我被迫直面生死这一课题，重新认识生命，重新学习如何去生活。

这一过程的转变，更像是内心深处的一场质问——在追名逐利时，我是否太过轻视自己了。考虑了父母、考虑了前途、考虑了金钱，却把自己生命的价值给简化成了外在的物质成就。

我意识到， **在这场生命的赛跑中，我也不过是个普通的个体，试图用有限的时间追赶无限欲望；用看不到的健康为赌注，去赌一个不确定的明天。**

写下这些感慨时，我的肠胃蠕动剧烈，不知是因为饿的，还是因低沉情绪无处排解。我和青青相约，今年回老家，一定要把那个免费的妇科检查给约上。与其一个人恐惧，不如两人一同面对。
