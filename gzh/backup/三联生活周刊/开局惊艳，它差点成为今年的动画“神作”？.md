# 开局惊艳，它差点成为今年的动画“神作”？

这个小镇出了问题。它不是像《寂静岭》那样的危机，也没有浓雾弥漫，但住在这座偏远乡镇的居民们正在陷入某种形式的疯狂。漩涡几乎吞噬每个人的生活。这部由Adult
Swim 改编的《漩涡》动画已经酝酿了很长时间，自
2019年首次宣布以来，被多次推迟，粉丝们一直在猜测这部作品到底发生了什么，直到今年九月底才终于与观众见面。**黑白色调、压抑的氛围、细腻的线条，《漩涡》以其独特的风格将观众带回到了伊藤润二漫画中的诡异世界。**

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/mscgUN7TcTIv2ZFC3mqsK2M3GsJarHQkQqicwGoncqkzibfkRCJhWWNBCcOENwMicqO5q13YbMOkRjlicrh1kO5M4Q/640?wx_fmt=gif&from;=appmsg)《旋涡》剧照

故事从女高中生五岛桐绘的男友斋藤秀一从临镇回到他们居住的黑涡镇开始。这个看似平静的小镇，突然被一种神秘的力量侵蚀，风开始以诡异的方式旋转，河流不再向前流动，而是形成了令人不安的漩涡。甚至人们的行为也开始变得古怪离奇。秀一的父亲首先被漩涡迷住，逐渐陷入对这种形态的病态迷恋。他不再满足于欣赏自然界的漩涡，而是将漩涡的形状强加于一切，包括自己的身体。**最终，他也化为了一具漩涡状的怪物。**

![](https://mmbiz.qpic.cn/mmbiz_png/SG64904nEXz1icZ9cLMX1fzRBaoqj50HEr7zIatH7Ovr9Em45PZSibNDrfJez4whNkdBulseD68xle3roP6pgUrw/640?wx_fmt=png&from;=appmsg&tp;=wxpic&wxfrom;=5&wx;_lazy=1&wx;_co=1)

《旋涡》剧照

桐绘也是如此。她的长发仿佛被赋予了某种诅咒，野草般疯狂地生长，形成巨大的旋涡，将她紧紧缠绕，吸取她的生命力。头发的生长不再是正常的自然现象，而是一种邪恶力量的蔓延，不仅不受桐绘控制，甚至反过来控制着她，令她的形象逐渐变得扭曲。**渐渐地，漩涡不仅侵蚀了秀一父亲和桐绘，还逐渐吞噬了整座小镇。**

恐怖、猎奇、惊悚，伊藤润二的名字早已成为恐怖漫画的代名词，在世界各地都收获了大批狂热粉丝。第一次接触伊藤润二的作品时，我还在读初中，租书店里那套精美的漫画深深吸引了我。女主富江的完美设定和紧凑的剧情，把我带进了他所创造的惊悚世界。从那时起，我便无法自拔地爱上了他的作品。

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/GDcWrI00HnUV52YzvBictxhPs7tZ6zDmxnh0vic3v4mGYhMAia4C4oVrI71bmoROiaU3a3Qs1wy2RLJxp7JDD16Uew/640?wx_fmt=gif&from;=appmsg&wxfrom;=5&tp;=webp&wx;_lazy=1&wx;_co=1)

《旋涡》剧照

**《漩涡》是1998年开始连载的长篇，迅速成为伊藤润二最具代表性的作品，并被多次改编为影视剧呈现于观众眼前。**
在《漩涡》中，恐怖的来源并非传统意义上的怪物，而是那种深藏于生活中的扭曲感。它无处不在，似乎没有边界，却不断侵蚀每一个看似平静的角落。那些生活中温暖而熟悉的存在，变成了不安的源头，逐渐将人们的生活转变为一场压抑而难以理解的梦魇。

**伊藤润二作品的最大特点之一在于他善于将普通的日常与不断侵蚀的恐怖完美结合。**
他的作品既没有过多的血腥场面，也没有让人不适的突然惊吓，而是通过潜移默化的各种生活中最常见之物，让恐怖逐渐扩展到读者心中，使读者在熟悉的环境中感受到难以名状的不安。

![](https://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pOP3TwNMZ6gVesFUy1vFnlq2PsLicBzudvoNLcmh7TGZTQmARu5yLBFpLrRs7W6M6T5hHPjk0bEX4Cw/640?wx_fmt=other&from;=appmsg)

《旋涡》剧照

**《漩涡》的成功在于它对“不可抗拒的力量”的刻画。这种力量不仅是自然界的某种形态，更是一种象征，一种对人类生活的侵蚀和控制。**
伊藤润二认为，恐怖并非仅来自鬼怪或超自然现象，而是源于人类内心深处复杂而难以控制的情感。当对某种事物的追求与痴迷超越了理性，便会演变为一种近乎毁灭的力量。**《漩涡》中居民对漩涡的迷恋，正是这种情感的象征，最终将他们推向了不可挽回的深渊。**

![](https://mmbiz.qpic.cn/mmbiz_gif/SG64904nEXz1icZ9cLMX1fzRBaoqj50HEM51gicq7Ff9MS6zg9xuXIMvdrmmzPxpBddeTysxW6smhb9LvEWRibrZA/640?wx_fmt=gif&from;=appmsg&tp;=wxpic&wxfrom;=5&wx;_lazy=1)

《旋涡》剧照

这在伊藤润二的其他作品中也有体现。比如富江，她是一位拥有令人难以抗拒的魅力的年轻女性，她的存在引发了周围男性的占有欲，最终这种欲望演变为不可控的杀戮冲动。她就像是一座深渊，吸引并吞噬每一个爱上她的人。她的存在似乎揭示了人类情感中极端矛盾的一面——迷恋与毁灭共存，爱与恨难以分割。

尽管包括《漩涡》在内的作品影响深远，伊藤润二的漫画改编历程并非一帆风顺。漫画的恐怖感建立在视觉静止的瞬间和氛围的细腻描绘上，而动画则要求连续性的画面和动态的叙事逻辑。**这种叙事媒介上的冲突，导致过去许多伊藤润二作品的改编都被认为是失败之作。**

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/mscgUN7TcTIv2ZFC3mqsK2M3GsJarHQklFcB1A7o6YRYZ5NTI8V6gVnjObajVW5fbQQnOu5hhZyjUlxQ9W5NeQ/640?wx_fmt=jpeg&from;=appmsg)

《伊藤润二惊选集》剧照

2018年，Netflix就曾与Studio
DEEN合作推出过《伊藤润二惊选集》，这部作品试图将伊藤润二的多个短篇恐怖故事串联成动画，但由于动画无法做到像漫画一样，通过细节的堆积和瞬间的定格令画作中的不安传递到观众内心，最终给人留下了“平淡无奇”的印象。

但是，为什么这次由长滨博史执导的《漩涡》第一集，却打破了以往伊藤润二改编作品质量低劣的魔咒？

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/GDcWrI00HnUV52YzvBictxhPs7tZ6zDmxDK6bUuTFlXxjVsmNuK4KZbkIhC8hc7eWDqtGgA05slnbyEBHvq996w/640?wx_fmt=gif&from;=appmsg&wxfrom;=5&wx;_lazy=1&wx;_co=1&tp;=webp)

《旋涡》剧照

在很大程度上，这与其忠实于原作艺术风格和叙事手法的改编策略有直接关系。由Production
IG制作，长滨博史（《虫师》《恶之华》导演）执导《漩涡》第一集，不仅保留了原作的黑白素描风格，还将它升华到了新的层面。为了使观众体验到漫画中那种无形却致命的恐怖，动画采用了大量2D、3D与旋转描线的混合技术，营造了一种令人不安的画面感，成功地将观众置于一种无法躲避的致命力量之中。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/mscgUN7TcTIv2ZFC3mqsK2M3GsJarHQkdhNgvW93hzlqXGR7LhCqt6pf00evXug1JRXS3jkL86Dzx0qLqSfGAw/640?wx_fmt=jpeg&from;=appmsg)

《伊藤润二惊选集》剧照

在叙事上，长滨博史也极大地尊重了原作的节奏。**《伊藤润二惊选集》的失败主要在于，原本在漫画中精心设计的瞬间，被快速的情节推进所取代，恐怖的逐渐积累感消失殆尽，留下的只是缺乏深度的惊悚元素。**
而《漩涡》动画在叙事方式上与漫画原作高度一致，采用了章节式的结构，每一集讲述一个完整的小故事。这避免了过于紧凑的节奏，让恐怖有了足够的时间慢慢渗透到观众心中。

![](https://mmbiz.qpic.cn/mmbiz_gif/SG64904nEXz1icZ9cLMX1fzRBaoqj50HEYDrNkjUyez7jrccgXDfZiaHYiaqCwichVnYrTYukz0MmwGVicYaib2udaibw/640?wx_fmt=gif&from;=appmsg&tp;=wxpic&wxfrom;=5&wx;_lazy=1)

《旋涡》剧照

姑且不论项目原因导致该系列后续几集质量参差不齐的问题，不得不承认，长滨博史对《漩涡》的诠释，是一次充满勇气的尝试。那种将原作中所蕴含的诡异与阴郁毫无保留地展现在银幕上的执着，为我们塑造了一个既熟悉又陌生的恐怖世界，仿佛从漫画的页面中流淌而出的幽暗气息，缓缓爬进了我们的日常。

在故事的最后，当小镇被漩涡彻底吞噬，秀一和桐绘的命运也在螺旋的力量中走向毁灭时，观众也不禁会思考，什么是我们生活中的漩涡？或许每个人心中都有一片无形的漩涡，代表着我们无法掌控的欲望与执念。即便是看似普通的美好和对生活的追求，一旦失控，就会变成吞噬一切的力量。

![](https://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pOP3TwNMZ6gVesFUy1vFnlq2ejbliaJQtf6ePu0MPlQKcdbib9fur03YpEoO1ia63liaiaUAibrAJpJLxbSw/640?wx_fmt=jpeg&from;=appmsg)

《旋涡》剧照

这或许也是几十年过去，《漩涡》仍旧如此令人难以忘怀的原因。它不仅仅是一部恐怖作品，更是对人类心灵深处恐惧的深刻探讨。当漩涡最终将我们卷入的时刻，或许我们才会发现，**一切的恐怖并非突如其来，而是早已悄然生长在我们每个人的日常生活之中。**
