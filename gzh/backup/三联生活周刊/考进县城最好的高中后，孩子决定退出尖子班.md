# 考进县城最好的高中后，孩子决定退出尖子班

文｜读者：绿叶子

当儿子跟我说他想去普通班时，我的心跳漏掉了一拍。那之前，我还沉浸在孩子考进县城最好高中尖子班的喜悦中。几秒钟之后，我对儿子说：
**“可以，你自己的感受最重要。”**
提出这个想法前，儿子内心一定经历了很大的冲突。他原来一直很期待考进尖子班，但中考前一个月突发胃溃疡出血住院，因为生病节点特殊，大夫允许我们在中考前出院，但中考后必须马上继续治疗，大夫还强调，这中间如果胃再次出问题就先别参加中考了，所以我们当时甚至已经做好考不了试就去私立高中的心理建设。我们在病房里查的分，觉得分数挺高，后来横向一对比，推测尖子班应该没问题了。那时候，孩子爸在背后和我说，孩子去普通班就好，省得太累。我了解他这个人，他这么说一方面是觉得孩子身体最重要，另一方面是想隐晦地表达骄傲——
**儿子考得好怎么选择都行。我说咱们怎么想不重要，孩子想法才最重要。**
胃溃疡伤口已经痊愈，但儿子还面临中度缺铁性贫血的问题，出院后需要补铁，好好休养一个假期。没想到尖子班需要提前报道，报道那天距儿子出院也才十天。此外，第二天尖子班还要提前上课，晚自习到九点五十。我们一听就觉得很累，在儿子的同意下跟老师申请不上晚自习。报道当天回到家，我们跟儿子说，
**白天如果上课觉得累也可以随时回家休息，身体健康最重要，能学啥样算啥样。**![](https://mmbiz.qpic.cn/mmbiz_gif/VkpaUkchBmW74riaUoDdKqZnXIEt47iaDsxZo9eR6iclH1CtfUvGZpzzice0akZ7LLaIIz1tSllEWrib38HUEyR60TA/640?wx_fmt=gif&from;=appmsg)《追光的日子》剧照第二天白天，我为儿子上学做准备的时候，他告诉我这个决定。就如其他家庭大事一样，我给孩子爸打了电话。电话那边的他语气变得凝重：“你跟孩子好好唠唠，等我回去也跟他好好唠唠……”作为一个喜欢出去和朋友喝酒唠嗑的东北男人，他始终坚信没有什么是“好好唠唠”解决不了的，如果一次不行，就两次、三次……如果一个人不行，就两个人、三个人……挂了电话，我知道好好唠唠的对象其实不应该是儿子，而是他爸。在
**漫长的育儿经历中，有多少次矛盾冲突，就有多少次妥协，** **他心里没数吗？为什么还要屡败屡战迎难而上。**
儿子从小就有主见，且倔强得很。小学四年级就确定了自己想上的初中，他想上户口所在地的乡镇中学。按照本地规定，我们也可以读住房所属学区的地级市初中。![](https://mmbiz.qpic.cn/mmbiz_jpg/VkpaUkchBmW74riaUoDdKqZnXIEt47iaDs96Lg1AjWyNRCQxnmqMTUy1FRhFficlFyY7kIiaF3ok3n1VLdxUiapiak3w/640?wx_fmt=webp&from;=appmsg)《小欢喜》剧照当年，同小区朋友家的孩子上初一，儿子还像以前那样常常找小伙伴玩，结果每次去找，人家都去补课了。儿子的犟劲儿上来了，一天晚上九点多去按门铃，结果那孩子补课还没回来，当时儿子就说他肯定不去市内初中了。真正小升初的时候，身边很多人都劝我应该去市里的初中，这样才能考更具竞争力的地级市高中。一个要好的朋友特意发微信跟我探讨这个问题，我解释得已经累了，后来就说尊重孩子的看法。那时候我是拿这个当挡箭牌，我也希望孩子念乡镇初中，因为我了解那的老师并不差，环境不那么卷，儿子会相对轻松，我也会相对轻松。但别人会说，孩子还小，不会考虑长远，大人应该考虑的，不能什么都听孩子的。
**可我觉得孩子再小也有自己的感受，尊重他的选择，他也会有学习的内驱力，如果替他做决定，可能会破坏孩子的自主学习能力。**
事实确实如此，儿子知道自己学，课堂上认真听课，课下作业从不糊弄。他不补课，也不会多做额外的习题，他说生活中不能光有学习，业余时间喜欢玩电脑游戏。能自由使用电脑的一个好处是他不会半夜起来偷着玩，所以每天能保证十点前上床睡觉，初三的时候也不例外。
**当然，轻松是相对的，压力也是有的，只是他不愿意承认。**
疫情的那个漫长假期，他完成学校布置的作业后就把时间都花在电脑游戏上，后来一开学就补了上个学期的考试，公布成绩那天，他说心里一直担心来着，我听着又好气又好笑。那是他难得跟我说感受的时候，不过排名出来了，他并没有掉下去。他逐渐也做到心中有数，保持着名次。![](https://mmbiz.qpic.cn/mmbiz_png/VkpaUkchBmW74riaUoDdKqZnXIEt47iaDsBq71v5qCktQ5qfe6wfv2IBqlnTwyjDctBdAzUn8uiaov9Scq53VbOZA/640?wx_fmt=png&from;=appmsg)这么心里有数的孩子，我们却总想要求他更好，真是惭愧。我们大人对自己都没有这样高要求，学生时代，我和孩子爸学习都很一般。现在，孩子爸的业余时间大部分花在喝酒和钓鱼上。
**我也是，本来应该在事业上升期时选择了相对轻松的工作，美其名曰听从自己的心。为什么刚刚出院的儿子想要轻松那么一点点，我们却难以接受？**
孩子爸回来了，一进门就说他也想明白了，支持孩子的想法。儿子也松了口气。无奈县城小，很多人在社交软件上发了尖子班报道时的盛况，再加上我之前嘴欠说过孩子考上尖子班了，当亲朋好友问起，我只好一一解释说我们现在请假了。关系亲近的才实话实说。很多人都告诉我，无论上尖子班还是普通版，学的东西都是一样的，都不轻松，如果这样的话，不如去资源更好的尖子班，这导致孩子爸更加耿耿于怀孩子的选择，我的心思也蠢蠢欲动。于是，我们又小心翼翼提醒儿子可以去试试，晚自习都不用上，白天累了随时回家休息。儿子听了没有发怒，但那神情很伤心，他觉得不上晚自习压力更大。
**我们劝他不用跟别人比，但其实这话很苍白无力，尖子班的环境里，怎么可能不比呢？**
以前听说某个学生主动退出尖子班，周围人都不理解，当时我还自以为能共情那孩子，知道自己的感受，有主动选择的能力是非常难得的品质。
**没想到这事放在自家孩子身上时，我却没能真正做到理解他。**
更何况，他是一个不久前生病还在康复中的孩子啊。明知压力和情绪会影响消化系统，那时就好像被执念操控了一样。这么想过之后，我又开始心疼儿子了，他是经历了多大的内心困扰啊，面临自己的内心冲突，还有家人的外部冲突。如果我不站住他这一边，他会是多孤独和难过。![](https://mmbiz.qpic.cn/mmbiz_jpg/VkpaUkchBmW74riaUoDdKqZnXIEt47iaDswFb1AgkXndpaOJZJ4GeRbr4K4zAB5pZWnV4icwFJtdcYKnPfVdRrPzQ/640?wx_fmt=webp&from;=appmsg)《沙海》剧照
**作为父母，孩子的出生最初是为了满足我们自己的需求，我们对他的很多要求，也仍旧是为了满足自己的期待。**
所以，孩子不欠我们，也不需要觉得对不起我们。这一次，我真正站在孩子这边，打心里真的认同他的选择。我告诉他一切都听他的，我们不再提这件事了。其实，我之前还有一点隐隐的担心，如果开学后他连普通班都不想去怎么办？
**现在想，不去就不去，如果他想休息也没关系，想休息多久都没关系。**
