# 年过五旬的妈妈，靠自学英语变现

“囡囡，楠楠妈来了，要给两千，拉扯了几个回合留了一千。”点开微信，看到妈妈刚发来的消息，我的第一反应是，“这是真的吗？”短短的一句话我看了又看，大脑才真的接受了这条信息。骄傲和欣喜溢满了胸膛，回想过去种种，又几乎红了眼眶。妈妈是在我大三那一年开始自学英语的，到现在已经七年多了。高中文化的妈妈，读书时选的理科，但她更擅长的还是文科。妈妈常和我说她的作文如何被老师当作范文在全班朗读，也常常追忆高中时曾给她写过情书的小张、小左、小眼镜、小平头们，用一种有点后悔但又带点骄傲的语气说：“假如当时我没有谈恋爱，肯定考上大学了！”![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz6z7Eje2hJibuEI8c7CCsoSSKXibkdAqBX9Zm5LdHCGnZwaRhNz61oe06vXdOc9icmbYicyPMjP5iaJddg/640?wx_fmt=jpeg&from;=appmsg)《小舍得》剧照妈妈和爸爸结婚没两年，爸爸上班的工厂就发不出工资了，家里有嗷嗷待哺的我，姥姥家还有很多需要帮衬。恰好有在广州发展的亲戚给爸爸介绍了一个工作机会，为了生计，此前26年从未出过山西的爸爸背上了行囊南下。为了照顾我的生活和学习，妈妈并没有和爸爸一起去广州，而爸爸只有每年过年在家待20天左右，从此他们开始了长达15年的分居生活。****这种情况一直持续到我去南京上大学，妈妈终于卸下了肩上抚育的担子，选择去广州和爸爸一起生活。爸爸在公司附近的城中村租了一间约十平米的房间，带一个简易的卫生间，没有淋浴，阳台上架个桌子和煤气灶就当作厨房。条件简陋了些，胜在价格便宜，每个月只要200元的租金。
刚开始，妈妈也找了一些工作。第一份工作是在出租屋附近的一个医院做保洁。妈妈需要在早上8点医生护士们正式上班前，将所负责区域的卫生打扫完毕。其实工作内容倒不算重，但做了两天，妈妈就说她不想去了。因为她需要进入医生护士们的办公室打扫，有的护士看到妈妈会表现出不加掩饰的嫌弃，也不愿让妈妈随意挪动办公室里的椅子等进行打扫。没有得到应有的尊重，妈妈觉得很委屈。妈妈的第二份工作是在小作坊里当车线工。****广州的城中村里分布着大大小小的手工作坊，大批大批的箱包、服装等经由这里发向全国各地甚至国外。妈妈找了一个做户外背包的小工厂，每天早上7点到晚上10点，一个小时10块钱，做了很长一段时间。工友们大多来自四川和湖南乡下，待人友善。大家做工时聊聊天，嘻嘻哈哈的，每天时间虽长，却也有其中的乐趣。![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz6z7Eje2hJibuEI8c7CCsoSSQ0ibGeCzgygkqc2twLiajOtoibticYpGAr0dvlicQV0Wz3HSGZ2CW4DZ54A/640?wx_fmt=jpeg&from;=appmsg)《小巷人家》剧照我大二时，妈妈有大半年没去广州，而是自己住在山西的家里。或许是因为一个人在家还是太寂寞和无聊，后来还是又去了广州。**这次妈妈没有再找工作了，而是打算在家做好一日三餐，把她和爸爸照顾好。**
十几年异地生活的影响慢慢显出端倪，两人虽然已经是二十多年的夫妻，但还需要很长的时间来磨合。之前每年就在一起不到二十天，还是过年期间，每天总有事情要忙，虽然也会吵架，但要做事时也就过去了。但现在每天要日常相处，两人不合拍的地方也渐渐显露出来。爸爸是个慢性子，平常话少，不会说甜言蜜语，长年的独居生活也让他缺乏照顾他人的能力。妈妈性格外向，遇见谁都可以聊两句，家庭和生活也让她总是把家人放在第一位。但两个人过日子，总要一个人来照顾另外一个，难免会有些委屈，如果被照顾的人能说些甜言蜜语提供情绪价值，或许也还能平衡，但可惜爸爸不是这样的人。两个人就经常吵架，一吵架，爸爸也不说话，回家吃了饭就去公司上班，留下妈妈在出租屋，一股火气不知该往哪里发。冷战时，爸爸白天在公司，可以靠工作转移注意力，也可以和同事们聊聊天满足社交需求。但妈妈就不一样了，在广州，她除了爸爸谁也不认识，爸爸是她每天唯一一个可以面对面交流的人。她没有亲人，也没有朋友，只能自己呆在阴暗逼仄的出租屋里生闷气。那段时间，妈妈总打电话向我抱怨和爸爸的冷战，我想如果妈妈有了可以专注去做的事，就不会整天把注意力都放在爸爸身上了，而现在网络这么发达，她完全可以自学点什么东西。那段时间正好我也在练习口语，觉得英语作为一门语言，入门门槛相对较低，而且在学习过程中可以了解国外的一些风土人情，也有很多人用影视剧和音乐当作英语学习资料，那学习过程会比较有趣，不那么枯燥。于是我向妈妈推荐学习英语作为爱好来打发时间，**妈妈的英语水平停留在那个年代的高中，相当于现在的初中水平，当时的学习条件本就不好，加上已经有近二十年没有碰过英语了，这个开头还是很难的，不过好在妈妈坚持了下来。**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz6z7Eje2hJibuEI8c7CCsoSS5tro8dUsuM4GTdKWWIbicdD5I5FkO4LRm8ICoqB2r5UGCVEYhPrSiaSw/640?wx_fmt=jpeg&from;=appmsg)《何时是读书天》剧照我把自己当时的学习资源分享给了妈妈，还给她买了五年中考三年模拟。**买五三的时候，一种奇怪的错位感产生了，我有点骄傲，也有点窃喜。**
骄傲的是妈妈在这种情况下愿意通过学习来充实自己，窃喜是没想到有一天我也会给妈妈买教辅资料。**自此，妈妈开始了至今七年的英语学习旅程。**
从那以后，每次打电话，妈妈都会分享自己的学习心得。先说学习英语好难，文章半天还没背下来，还担心发音也有问题，音标不知道读的标不标准。我会劝慰和鼓励几句，妈妈肯花时间来学习已经非常了不起了，而且英语学习本身就是一件需要积累的事，不要着急，慢慢来。然后妈妈就会话锋一转，对我的话表示赞同，然后又开始开心地向我列举最近学习取得的进步，比如原来完形填空全都错，现在能对一两个了云云。大三时，妈妈又回到了老家居住，不过这一次有了英语的陪伴，妈妈不再说自己孤独和无聊了。妈妈每天会给自己规划学习时间，上午九点到十一点，下午三点到五点。寒暑假时，我在一个房间学习，妈妈在另外一个房间学习，偶尔出来上厕所，听到妈妈在大声的朗读和背诵，我也会倍受激励，自己学习也更有劲儿了。大姨，也就是妈妈的大姐，是她们四姐妹里唯一上过大学的，现在在深圳定居，退休之前做过老师，做过大企业公司杂志的主编。虽然身在深圳，但她一直挂念着老家亲戚孩子们的学习。作为一个从小地方走出，在大城市拼搏并定居的人，大姨深知教育的重要性。因此在我研究生阶段，大姨组织给老家孩子们进行线上学习辅导，并且把妈妈也拉了进来，做一个“助教”。妈妈既跟着孩子们一起和大姨学音标发音，也承担了辅导孩子们语文的任务。![](https://mmbiz.qpic.cn/sz_mmbiz_png/XnMeqb0xcz6z7Eje2hJibuEI8c7CCsoSSOwkfWXHhG9ZWS2sdDxruSa7uRu8VCqjVaJOlOm5rc7PtoCBxoXhu5Q/640?wx_fmt=png&from;=appmsg)小猫陪伴着妈妈学习（作者供图）每天晚上7点到8点，妈妈都在辅导孩子们。那段时间我们打电话的内容就变成了，这两个孩子最近又不好好学了，给他们布置的任务一个都不做，或者是某某最近的进步很大，上课会声音洪亮地、情感饱满地朗读等等。抱怨时我就劝她不要着急，小孩子玩心大很正常，听到有进步时我也跟着一起高兴。有好几次，妈妈都打电话来，语气非常生气，说自己再也不带这两个孩子了。然而过了两天，孩子的奶奶又来央求，她又狠不下心，直说着：“我就带到中考结束！”就又“重拾教鞭”了。就这样过了一年，两兄弟中考都考上了市里的重点高中，妈妈也就暂时卸下了肩上的担子。妈妈会在家大声的朗读和背诵英文，有时邻居来串门，也会碰到妈妈在学英语或给孩子们上课，慢慢地，村里人都知道妈妈在家学英语。对于这么大年纪了有时间居然不休息不打牌而是在学可能一辈子都用不上的英语，他们大多数人都不理解，有一些人还会带点嘲笑的意味打趣：呦，你这是要考大学了！学的这么认真！妈妈一开始还会觉得不好意思，后来想明白了，再有人这么说她就回答：可不是呢！以后出国还要用呢！今年暑假刚开始没多久，邻居赵大妈想让即将上初二的小孙子来补习英语，说是试了好几个培训班了，都没有效果，于是妈妈接下来了这个活儿。我问妈妈有没有信心，她声音洪亮地回答：有！我现在的水平，教个初中还是没问题的！又过了两天，上了第一节课，我问起妈妈的教学感受，她说这个小孩的基础实在太差了，不过好在愿意开口读，说明是有学习意愿的，那就有希望。就这样，妈妈给楠楠补习了差不多一个半月。期间她经常跟我分享“教学心得”。我也有一些顾虑，比如妈妈并非教学科班出身，这孩子找补习班都没用，妈妈补课会有用吗？别白费了力气，还没有效果，最后还影响了邻里关系。妈妈告诉我，楠楠跟她说，补习班的老师讲得太快了跟不上，在这里是一对一指导，而且妈妈也是自己摸索着学习的，对孩子来说，可能比“学院派”的补习班老师讲得更容易理解。![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz6z7Eje2hJibuEI8c7CCsoSSIAPjha9qNEsxd5h6G0oukC0x1PUpf3lYuKwygS31SVuNL2r5BgxD9A/640?wx_fmt=jpeg&from;=appmsg)《小舍得》剧照暑假渐渐到了尾声，对于这次的补习要不要收费，我和妈妈也讨论过。虽然赵大妈开始是说了是有偿的，但一来妈妈之前没有给人补习过，不知道效果会怎样，二来也算是自己锻炼一下，所以我们是没有计划要收费的。想着这次当作“新店开业大酬宾”，如果真能行，下一次再收费。**没想到楠楠的父母还是坚持要给，还约定了开学后每周六周日楠楠还来上课。**
楠楠开学没几天，妈妈兴高采烈地打电话来告诉我，楠楠在开学的英语测验中，分数比之前提高了近二十分，孩子和家长都很高兴，都称赞是妈妈的功劳。不仅如此，或许是看到了效果，又有别的邻居找上门来，妈妈又收了一个“弟子”！现在每到周末，妈妈的“课时”安排得满满当当，虽然偶尔也有自我怀疑，不过大多数时间，妈妈都卯足了干劲，一再说着自己也要继续抓紧学习，提高水平，更好地辅导小朋友们。![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz6z7Eje2hJibuEI8c7CCsoSSLgtvXNdU8oqsRLzhtQ8YX8dVwcDibdzTo6gRqfI4oOicOv2VfsNKMeUQ/640?wx_fmt=jpeg)《我的前半生》剧照**回想起这一路妈妈的变化，做了这么多年的家庭妇女，为了家庭放弃了很多自我，学英语这事儿，终于让妈妈又重活了一遍。**
