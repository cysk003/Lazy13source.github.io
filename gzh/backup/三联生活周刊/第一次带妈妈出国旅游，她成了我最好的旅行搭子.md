# 第一次带妈妈出国旅游，她成了我最好的旅行搭子

文｜读者：渔渔

在从吉隆坡去马六甲的路上，年逾四十的华人司机阿彬聊起自己的妈妈，车上坐着的也正好是两对拼车的母女。“我们这儿的人觉得父母也有属于自己的人生”，阿彬说，他妈妈自他上大学后就不再管他，此刻或许正天南海北旅游。连他都摸不准她的行程，只有在飞机抵达机场时，妈妈会叫他来接机，或是在旅游地准备买衣服时打来电话：“儿子，你穿什么size？”
**“妈妈你看，你也可以像她那样”，那天我对妈妈说。**![](https://mmbiz.qpic.cn/sz_mmbiz_png/XnMeqb0xcz40jlibzwDrw17mZQu9GRajsRiapxrLAdAtsfVg9UTkkRZ086ypibyjYI2Sr0Hx2sibur3yRYiaW2b0qCA/640?wx_fmt=png&from;=appmsg)《二十不惑》剧照说出这句话，感觉仿佛26年的母女关系开始倒置，妈妈变成了我的女儿。我鼓励她走出去，就像多年前她把幼小又腼腆的我推到陌生的店铺前，让我独自去问路。妈妈今年52岁，从来没有独自出去旅行过。在互联网普及的时代，直到去年才学会用支付宝扫码支付。每次来我工作的城市看我，都需要爸爸给她在手机App上订一张高铁票。
**一年前妈妈退休，每晚我打视频电话回家，问她一天做了什么，她总说：“在家打扫卫生”。后来爸爸因工作变动调到另一个城市，她会时不时拿去换洗的床单，帮他做饭洗衣。而明明这些事，爸爸自己也可以做。**
后来读《明亮的夜晚》，发现书里的妈妈几乎也是如此：连跟朋友两天一夜的旅行都没有去过，唯一一次的出国经历是和爸爸一起去日本。有一次妈妈动手术，还担心爸爸一个人在家没饭吃，让女儿回家给他做饭。![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz40jlibzwDrw17mZQu9GRajs8XJqI0GicGA2XRlKSod2smqZnLhYGos8qqTLDsaGUwKz71FPe62xLOg/640?wx_fmt=jpeg)《小敏家》剧照为什么妈妈从来不独自出门？身边的女性亲友，大都就算可以独行也总要被带领，就算学了车也不敢独自上路。好像被带领是默认的，而独自出行从来不是一个可供选择的、和“与他人一起”并排的选项。
**好在那段故事的结尾是妈妈突然跟女儿说，她要跟着自己曾经是同事的朋友去墨西哥玩。作者写，去墨西哥这件事对妈妈来说“是一场革命”。**
“一场革命”。对于从小镇出生、长大的我来说，出国也曾是遥不可及、从未设想过的选项。等到工作后终于有了时间和金钱，我跃跃欲试想要验证独自出行的可能。还要带妈妈一起。第一次出国，这场关乎创造的旅程我们必须一起书写。今年五一，妈妈说，那就去吉隆坡吧。一个不算遥远却充满异域风情的城市，承载了我们对远方世界的想象。
**这是我第一次单独和妈妈旅游，我们好像调换了身份，小时候她扮演“照顾者”的角色，而现在玩不转智能手机，更不懂英文的妈妈只能依靠我。**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz40jlibzwDrw17mZQu9GRajsS8u89Px5yMoUFhNR4umhNcmgtZTP82T8k3BtRvCUEpL767ZdyHuPWw/640?wx_fmt=jpeg&from;=appmsg)《妈妈》剧照我提前二十多天开始查攻略、做计划，在Airbnb上订到一套两室一厅的民宿。民宿在市中心，40层，阳台透亮的玻璃窗直抵地面，窗外像“大哥大”一样的地标性建筑竖起高高的天线，仿佛近在眼前。在51层的楼顶，还有无边泳池。站在那个角度看吉隆坡，有种疏离的陌生，又觉得自己是能掌控整座城市的主人。我以为妈妈会为我定的民宿感到满意，但她没什么反应。并在入住后不久从白色被子上捻起一根毛发，说：“还是该订酒店，民宿的床单被罩肯定没换洗过。”听起来好像责怪。住进好房子的喜悦一瞬间被冲淡。这样的话语，在她刚到我租住的房子时我就已经领教过。花洒的水压太小、房间太窄、厕所和厨房的洗手池只能共用、靠近大路边太吵......她说了一遍又一遍。可那已经是我能找到的最好的房子了。她就这样把我找房子的辛苦、努力营造体面生活的付出都一票否决。她不知道的是，在期待她到来的前一个星期，我把床单被罩都换洗了一遍，擦了桌子，拖了两遍地。
**那种熟悉的感觉又回来了。还是那个扫兴的妈妈，从来不会给予我肯定的回应。但我总不死心，总想得到她的认可，结果每次得到的都只有失望。**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz40jlibzwDrw17mZQu9GRajsEOXicB8dibmP5Y1ia8FGM1r6mHYeUz4SrvZREFib9QV4aQNDRJUdSqmPcw/640?wx_fmt=jpeg)《酒鬼都市的女人们》剧照以前我不会反驳，但在那一刻或许是潜意识里知道现在的我比妈妈更强大，决策权在我手里，气话脱口而出：那有本事你自己来订酒店，不然就别说话。她似乎意识到了什么，那天睡觉前，突然对我说“谢谢”。
**我有点内疚，觉得自己很残忍，仿佛是在向手无缚鸡之力的人施以暴力，明知道对方无力反抗。但在这次尖锐冲突之后，旅行突然渐入佳境。**
妈妈开始变得温和，不再干涉我的决定，无论我带她做什么，都投入其中。我不是个能把所有事情都安排妥当的女儿。我们曾顶着吉隆坡34度的高温走了很久也找不到导航上的饭店，也曾被小红书上“双子塔最佳打卡机位”欺骗，跑了很远结果一无所获。但她都没有抱怨。五一假期很短，飞机往返就耗费了两天，在吉隆坡的日子我们总是在路上，早晨驱车去郊区印度教圣地黑风洞朝圣，再到市区的茨厂街、中央艺术坊闲逛，再穿过生命之河，抵达独立广场，晚上去繁华的武吉免登感受人潮；第二天途经布城到马六甲，登上圣保罗大教堂所在的山坡眺望马六甲海峡，游船欣赏完这座充满历史的小城，回家已几近凌晨。![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz40jlibzwDrw17mZQu9GRajs1QcticRKB35basibBfyhaSdyTU7LC1daBKTNibFkB0ZjDDbCLcrIQxjSQ/640?wx_fmt=jpeg)妈妈在独立广场（作者供图）很少有朋友能跟上我通常日行两万步的特种兵行程，但妈妈可以，大概我体能好的基因也遗传自妈妈。没办法松弛地旅游，或许也是在过去拮据的童年生活里她潜移默化教给我的，我们只要出门，就是在花钱，每一分都要花得值，所以一刻也不能停下。好在这次她一句也没提钱的事儿，不再计较和心疼花钱，我一直紧绷的心也放松了很多。
**但途中，我越来越能感觉到那个曾经我眼里无所不能的人已经变得弱小。走到一处景点，妈妈提议要帮我拍照，但等我拿过手机一看，全是闭着眼、表情怪异的照片。见我脸色拉下来，她像做错事的小孩，咕哝说手机上的人脸太小，看不清我的表情。**
原来妈妈已经开始老花，手机上的字开始变得模糊，那些对我来说轻而易举的事都已经成了她独自出行的阻碍。“没关系的”，我鼓励她多拍，妈妈其实很有拍照天赋，除了表情，构图总是完美。而阻碍我出国的原因是什么呢？是羞耻于自己英语太差。高考时，我的英语考了一百四十多分，但到上海上大学后遇到本地同学，才发现其中的巨大差距。每次上英语课，都自卑得想逃走，默默祈祷老师不要叫我起来做英语对话。![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz40jlibzwDrw17mZQu9GRajsRUsIQwQ4mR5WRWjN2W82SQMxhMOgQZ4a7qTia4n8wMsAF0v5Tfn8UmQ/640?wx_fmt=jpeg)去清真寺，妈妈和我都穿上了传统的袍子（作者供图）在吉隆坡，我终于把自己丢进语言不通的环境，硬着头皮去沟通、交流。我们去一家印度餐厅吃一种印度特色卷饼，饼擀得比一只手臂还长，烘烤得又薄又脆。点餐时我蹦出简单的词汇“long”，靠着手势比划，服务员立即会意。又有一天晚上，我们去一家餐厅吃晚饭。店里已经坐满了人，我问“have
some
seat？”对方跟我说不止他们店面前的位置可以坐，旁边店的也可以，点完菜后他又告诉我，一会儿要自己来取餐，我已经忘记了他的英文原话，好像有什么“here”，我居然听懂了。
**看着一张全英文的收据从机器里滑出来，突然觉得好神奇，明明我连一句话都说不通顺，但别人却知道我想表达什么。**
原来最大的障碍只存在于心里，十几年来关于开口的恐惧好像在一瞬间被打破，哪怕只是在最简单的点餐场景。那一刻我压抑住雀跃的心情，仿佛发现了世界有一套超越语言的通行法则，这个秘密是给勇敢之人的奖赏。而在刚踏上旅程时我们还很紧张。订票时就知道航班不提供餐食，但上飞机后看到有乘务员给别的旅客送餐，才知道有付费餐食。可我不知道在哪里订、怎么支付，也不敢问。
**4个小时的时间里我们默默啃着自己带的馒头。后来才发现，座椅背后的航空杂志里有一页就是菜单。饭菜的名字是马来文，配有图片。**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz40jlibzwDrw17mZQu9GRajsDdFX1YggHAeAsGI3DyibCD51iaA0zsQaic5gIq5LbCRiasMeGpKnTNoRUw/640?wx_fmt=jpeg&from;=appmsg)作者供图等踏上返程的飞机时，我和妈妈都放松了很多，决定体验一次在飞机上点餐。反正还有现金，就算不知道怎么说，直接递钱总是没错。妈妈主动说：“我来点”。我很惊讶，没想到她相当自信：“这有什么难的，反正听你也是说‘this’‘this’”。原来我的所有举动妈妈都看在眼里，这些行为给了她很大的信心。
**这趟旅程，让我们都变得更有勇气了一点，发现打破那堵横亘在面前的透明的墙，世界并不遥远。或许将来妈妈真的能独自旅行呢，我期待有那么一天，她去经历属于她自己的人生。**
曾经我羡慕别人家的妈妈，高知、有钱、有见识，能够给下一代提供很多资源和支持。而我的妈妈好像都帮不上什么忙。从大学读什么专业到做什么工作，都只能依赖自己的选择，一路走得跌跌撞撞。刚到吉隆坡时，落地后要到机场的ATM机上取现金，试了好几次，ATM机上只跳出一长段马来文，却不吐钱。明明已经按照攻略上说的用银联卡、开通境外取款。虽然没有现金不至于寸步难行，但我当时还是急得冒冷汗。心里总有最坏的预设：该不会出国旅行的第一步就被困住。
**妈妈一直情绪稳定，提醒我要不就用最原始又最简单的办法——找人借钱。好在后面换了一个ATM机，终于取出钱来。**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz40jlibzwDrw17mZQu9GRajsNG32QxwAApLpYUjLicOacw2vBCnUrU7mwic3cxDn9TQXosX7DiaOWkAew/640?wx_fmt=jpeg&from;=appmsg)作者供图晚上近十点我们才坐上从机场去民宿的网约车，路程有1个小时，周遭漆黑一片，广告牌上裹着头巾的女士照片时不时一闪而过，不知是哪个代言明星。唯一的光亮来自远处楼房车库的灯，吉隆坡的车库都建在地上，有四五层，彻夜明亮。如果是我一个人，大概不敢在语言不通的国家踏上一辆陌生人的车。但妈妈在我身边，就生出一种巨大的安心。
**** **因为有妈妈在，我做到了很多事。我们还能一起做很多事。**
