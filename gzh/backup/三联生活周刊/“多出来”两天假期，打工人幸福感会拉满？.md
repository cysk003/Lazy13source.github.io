# “多出来”两天假期，打工人幸福感会拉满？

[**返回列表页**](/gzh/三联生活周刊)

***本文为「三联生活周刊」原创内容**
文｜邢海洋11月12日，《国务院关于修改<全国年节及纪念日放假办法>的决定》公布，明年的假期安排有了一个显著的变化，春节假期增一天，五一小长假加一天，全年公共假期从11天变为13天。**全国假期安排历史上，这是自2008年以来首次公共假期天数变动，也是仅次于上世纪九十年代单休变双休后的大变动。**
公共假期多两天，不能和一周休一天到休两天的飞跃相比，却是20余年来增长最多的一次。一年365天中法定节假日115天，再多两天，就是117天，看起来似乎没有多多少，但仅就公共假期而言，11天多了两天，变成13天，却堪称跨了一大步。  
**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz4kLtP4ofR5yf7P3PE61eYZK27yqlpUuTFNNTzC422BsYcPX5fsQicAldickpATBLbhzRYTk66ia186g/640?wx_fmt=jpeg
&from;=appmsg)**《我准时下班》剧照20余年间节假日安排理念和时间上曾有巨大改变的是2008年，此前全国公共假期是10天。在2008年的节日调整中，“五一”三天的长假减少为一天，多出来的这两天，又加上一天，共三天分别赋予清明节、端午节和中秋节三节。那是一个分散假日的安排，城镇居民有了更多的分散化休假的闲暇。**不过没多久，五一小长假还是以向前向后借周末的形式回归了，说明经济仍然依重脉冲式的旅游效应。**
季节上五一是一年中旅游旺季的开始，以一个全国性集中出行打开全年的旅游季节，似乎是一个颇有意义的开局。但五一“没假硬休”凑长假的弊端也是明摆着的，休息了五天，如果是在家里纯休息还好，若是出门旅行，时间上本来就紧张，又想用有限的时间踏访尽量多的景点，很可能演变为特种兵似的旅游。结果是五天行程匆匆，归来一身精气神全耗尽。可接下来的双休日又被借走，身体无法调整到正常的作息状态，工作和休息都耽误了。**“凑假期”的方式，打乱了原本规律的工作节奏，造成身体疲惫，反而降低效率。****![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz4kLtP4ofR5yf7P3PE61eYZ3qdNJI4UdiaUzeJY4aqmAK61IOWpUdAgSe5NpZMTIHoA7L9UfdfzGibw/640?wx_fmt=jpeg
&from;=appmsg)**《无法成为野兽的我们》剧照节假日分散、时间短助长了调休，调休越来越离谱，那些单休企业的员工，遇到长假调休一个月的休息都可能“泡汤”。**拼凑的长假“假象”，降低了公众的幸福感，引起越来越多的“吐槽”。**
这次5月2日再次被定为法定假期，也就避免了更多的调休。新的节假日安排强调调休后上班一般不超过6天，显示了政策制定部门的从善如流。新的假日安排，还有一项重大的变动。2014年全国节假日纪念日休假有一个让人匪夷所思的变动，除夕当天从春节中剔除，春节变成初一、初二、初三。而在此前春节的法定假期是除夕和初一、初二。有一种说法，除夕当天很多单位都会提前放假，将放假日延后等于将这一假日“隐形化”，是增加员工福利的做法。但除夕从假日里剔除，使团圆饭也名不正言不顺，还是让人摸不着头脑。中国人忙碌一年，最重视除夕的阖家团圆，团圆饭和守岁，团团围坐看春晚，样样都在除夕。正所谓“宁忙一年，不忙一天”，假日重回除夕，那些“站好最后一班岗”的企事业员工乃至政府公务员，回家团圆也不再背负道德负担了。**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz4kLtP4ofR5yf7P3PE61eYZbk8Dib1tmy7iaVoPAKAKwd8SySxLfVSRg5vgzAgHjRaMiaibJFjsFR34qQ/640?wx_fmt=jpeg)**
《小舍得》剧照**在公共假期安排上，此前的11天在世界190多个国家及地区中排名80+左右；增加到13天，排名超越了中国台湾，进入新的档次。**
不过在亚洲这个公共假期普遍偏多的地区仍处于中等水平，作为对比的是日本有16天的公共假期，韩国是15天。我国香港拥有17天公共假日，与内地相异的是：耶稣受难日，复活节、佛诞日、特区成立纪念日、重阳节和圣诞节。不难看出多元文化带来更多的公共假期，但同样遵循民族传统，内地并不休重阳节，这是和香港特区有区别的。**发达国家倾向于短假多休，带薪休假，尽可能地减少假期对社会运行的影响，同时差异化个人的休假时间，休假常态化，休假不扎堆，提升个人的体验。**
此次调整未增加节日，而是在两个既有假日上增加时间，走的仍是集中休长假的路径。何以如此？其实从对社会消费的拉动看，当假期加长时，人均日消费是下降的，2019年五一假期4天，全国国内旅游总人数1.95亿人次，人均每日消费是151元。2024年五一假期5天，全国国内旅游出游合计2.95亿人次，人均每日消费113元。不难看出，假期增长出行人数飙涨，但人均日消费“跳水”了25%。总体上经济拉动了，但具体到个人，旅游体验反而变差了。当然需强调的是，穷游也不完全与糟糕的旅游体验对等。及至国庆节长假，2023年中秋与国庆连在一起，凑足8天，日均消费114元，比2019年减少了3.9%。**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz4kLtP4ofR5yf7P3PE61eYZlU8Gt4WqLicfonEIhAoiaObKvFibap4qsviaN9Bj7ApLaNR5dyDibWuRbDg/640?wx_fmt=jpeg
&from;=appmsg)**《玫瑰的故事》剧照假期旅游相当部分的费用是交通费用，这个意义上，假期越长分摊在交通上的费用越少，是节省性因素。但假期同样是注重体验的，时间拉长新鲜感降低，对提升旅游体验、修养身心并无裨益。假期的安排需要兼顾多方面的考量。**有人说，集中休假是为了对抗很多单位年假成摆设，带薪年假得不到执行的情况。**
诚如斯言，人社部一项调查数据显示，有近半数的人没有享受到带薪年休假。按照在职职工工龄计算，我国人均带薪年休假约为10天，实际上人均享受带薪年休假天数仅为6.29天，民营企业职工甚至不足4天。![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/haySDMRphfkbLEdQkNovFWWkPLibRLgz9WBqkgicVZBX9uWEANq8jlPGSPiciaaJkRUiaw50Jrr3XVN8OaJwEgObhUA/640?wx_fmt=other&from;=appmsg&tp;=webp&wxfrom;=5&wx;_lazy=1&wx;_co=1)《没完没了》剧照**增加了两天公共假期，相当于政府强制多放假，弥补带薪年假的不足。**
但更为集中的放假，摩肩接踵人山人海的景区人流、酒店餐饮机票火车票的紧张，以及叠出不穷的消费“刺客”，都在显示更平衡更分散的假期的益处。其实对各行各业而言，当长假来临，旅游业独旺其他行业按下暂停键的情况未必对整体经济有益。增加了两天假期，减少了财富创造，提高了企业经营成本，靠旅游业未必能弥补回来。对个人，假期自然是多多益善。“点赞”“在看”，让更多人看到![](https://mmbiz.qpic.cn/mmbiz_gif/c2Sib3Mp7pON9hkSZwdTibRHNZSMPyiapUCHJwlyoZVBC3SfmPmF0VKjkm3NiaToQloHFJ6icyicqZnqgXp6pSQJt5gg/640?wx_fmt=gif&from;=appmsg&wxfrom;=5&wx;_lazy=1&tp;=wxpic)
