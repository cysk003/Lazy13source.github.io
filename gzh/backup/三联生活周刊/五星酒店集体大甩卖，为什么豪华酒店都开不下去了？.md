# 五星酒店集体大甩卖，为什么豪华酒店都开不下去了？

[**返回列表页**](/gzh/三联生活周刊)

公众号【懒人搜索】备份，仅供查阅学习

# ***本文为「三联生活周刊」原创内容**

文｜谢九

最近两年，国内的五星级酒店被大量拍卖，曾经火爆一时的五星级酒店热潮，正在迅速降温。**五星级酒店为什么越来越难以经营，背后又折射了怎样的经济现象呢？**
这场五星级酒店大退潮，从2023年就已经开始，直到现在仍在加速。由于酒店业主的资金链紧张，越来越多的投资方加速退出，很多知名奢华酒店，都成为被甩卖的对象。最近两年被甩卖的酒店，包括顶奢酒店上海宝格丽，以及经常接待美国总统的北京金茂威斯汀等等。而且，能够成功转手的豪华酒店只是极少数，还有更多酒店无人接盘，苦苦等待新资本进入。比如报价28亿元的北京索菲特酒店，报价15亿元的佛山希尔顿，报价5亿元的重庆悦榕庄酒店等等，这些挂牌拍卖的酒店或是无人问津，或是多次流拍……![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz5fxhFCnUz6NeuNa4ryf40px7alnP5mtiaX1W7YzWHe00IZicpfQ3OypoT8kfUP622L9lrTFibPJ7iaZA/640?wx_fmt=jpeg&from;=appmsg)《三十而已》剧照由于愿意接盘的资本越来越少，很多五星级酒店最终不得不选择关门倒闭。北京的长城饭店和瑜舍酒店，这两家老牌和新潮的五星级酒店，位于北京核心的燕莎和三里屯商圈，在2024年先后关门。**由于资本退出越来越多，五星级酒店数量最近几年明显减少。**
按照文旅部的数据，
2019年底，我国五星级酒店数量845家，截止2024年三季度，我国五星级酒店数量为736家，不到5年时间，我国五星级数量减少了100多家。大量五星级酒店正在悄然消失。**我国五星级酒店之所以快速退潮，主要原因在于房地产市场降温。而当年五星级酒店兴起，很大程度也是靠房地产推动。五星级酒店在中国的命运起伏，可谓“兴也地产，亡也地产”。**
曾几何时，我国曾经兴起一股五星级酒店的热潮，从一线城市到三四线城市，大量豪华五星酒店拔地而起。20年前，我国五星级酒店还不到200家，短短几年间就膨胀到了800多家。当年五星级酒店快速发展，地产商扮演了极其重要的角色。很多地方政府为了招商引资的需要，在拍卖土地时，会要求开发商配套建设五星酒店，以此提升地方形象。比如，前些年东莞市曾经挂牌一宗土地，就公开要求“竞得人须在上述地块内建设一家建筑面积不少于3万平方米、房间数不少于150间且不多于160间的酒店，引进世界知名度假品牌酒店管理集团，在该地块经营同品牌酒店15年以上。”类似的例子屡见不鲜。有些地方政府为了鼓励开发商建设豪华酒店，甚至还会给予一定地价优惠。![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz5fxhFCnUz6NeuNa4ryf40pKhZQ1uOiaMeBYiaFHMmNQgiaQhztXvjcam6LEz4YiaQPC7Oz3afibOL25tw/640?wx_fmt=jpeg&from;=appmsg)《坚如磐石》剧照对于开发商而言，在前些年房地产的黄金年代，土地是最宝贵的资源，为了获得土地，即使酒店自身不盈利，很多开发商也愿意配合地方政府建设豪华酒店。而且，豪华酒店还可以提升周边价值，也可以抵押融资，能带来很多看不见的好处。**过去很多年，房地产开发商就成为五星级酒店的主要玩家。酒店自身是否盈利并不太重要，即使酒店亏损，靠房地产的盈利也足以消化。****但是，随着这两年房地产市场快速降温，这一套打法开始玩不转了。**
由于房地产主业越来越不景气，甚至大幅亏损，以房地产利润养酒店的模式已经不再可行。尤其是很多开发商面临巨大的债务压力，需要变卖资产偿债，酒店资产就开始被大量拍卖。**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz6KT3TZiah2KNaGsFJTb2dpBAfGv6ibic7xH1iaq0rk8R1WN4eFxC9snmnSJ9UbPRo5Vsdtyib7nGsKCNQ/640?wx_fmt=jpeg)**
《三十而已》剧照豪华酒店和开房地产发商之间的关系，从富力地产的命运最能看清。2017年，当年万达面临流动性危机，断臂求生发起了世纪大拍卖，融创和富力地产分别接盘了万达的资产，其中富力地产出资199亿元收购了万达旗下的77家酒店。富力地产接盘后，成为全球最大的豪华酒店业主，巅峰时期拥有90多家豪华酒店，包括万豪、洲际、丽思卡尔顿、希尔顿等知名品牌。**但是由于地产主业恶化，过去几年，富力地产不得不陆陆续续卖出酒店资产，到了2024年，由于被债主淡马锡提出清盘，富力地产的68家酒店资产更是被全面接管。****![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz5fxhFCnUz6NeuNa4ryf40pcxLcMFsE3bq7ZYJKxFptZ7laJicQgRXfs8SkzY6roDqyH2Zsq15tAsQ/640?wx_fmt=jpeg)**
《寄生虫》剧照**除了开发商自身的债务压力导致豪华酒店被大量拍卖，酒店自身的经营压力越来越大，也是五星级酒店退潮的重要原因。**
最近几年，由于经济增速放缓，商务会议减少，游客消费能力下降，五星级酒店的入住率越来越低。以文旅部的数据来看，2024年三季度，五星级酒店的价格平均为599元，同比下降了5%，平均出租率同比下降了4%。**在收入放缓的同时，五星级酒店的支出却没有减少，特许经营费、管理费、人工成本以及物业维修和折旧等等都是刚性支出，很多五星级酒店都深陷亏损。**
对于房地产开发商而言，由于旗下的五星级酒店没有造血能力，甚至还因为亏损而失血，当债务压力来袭，也就无力继续维持酒店运行，最终只能选择转卖，甚至被强制接管。正是在这样的背景下，这几年国内豪华酒店纷纷改换东家，甚至关门大吉。以全球最大的酒店业主富力地产来看，在接盘万达的酒店资产后，酒店业务亏损额度一年比一年扩大，2018年到2022年，富力的酒店业务亏损从4亿多元增长到20多亿元，累积亏损超过60亿元，直到2024年，旗下60多家酒店因为债务问题被接管。![](https://mmbiz.qpic.cn/mmbiz_jpg/c2Sib3Mp7pONauDXUntJib4OxQFeZwIJHowqM3h8ygiaqk7EVvsHLSYbWtEqENDfJzS0dmr6hXmkpkmOdibWnkXSHA/640?wx_fmt=jpeg&from;=appmsg)《承欢记》剧照不只是富力，其他很多知名开发商，也都纷纷将豪华酒店转卖，比如上海顶奢酒店宝格丽，2年前被华侨城以24亿元卖给了一家水泥公司；深陷债务危机的碧桂园去年将华南地区最大的佛山希尔顿酒店以15亿元挂牌出售。在过去房地产的黄金年代，过度膨胀的房地产资本将触角伸到各个领域，制造了一系列虚假繁荣的泡沫，比如足球，比如五星级酒店。现在随着房地产降温，地产资本制造的各种泡沫，正在一个个破灭。“点赞”“在看”，让更多人看到![](https://mmbiz.qpic.cn/mmbiz_gif/c2Sib3Mp7pON9hkSZwdTibRHNZSMPyiapUCHJwlyoZVBC3SfmPmF0VKjkm3NiaToQloHFJ6icyicqZnqgXp6pSQJt5gg/640?wx_fmt=gif&from;=appmsg&wxfrom;=13&wx;_lazy=1&tp;=wxpic)
