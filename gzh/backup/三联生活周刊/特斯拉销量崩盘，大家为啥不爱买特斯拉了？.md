# 特斯拉销量崩盘，大家为啥不爱买特斯拉了？

***本文为「三联生活周刊」原创内容**

文｜谢九

电车巨头特斯拉，正面临有史以来最大的挑战。最近，特斯拉公布一季度销量，由于销量远低于预期，被华尔街分析师称为一场绝对的灾难，特斯拉的股价也随之闪崩，今年以来的累计跌幅已经超过了30%。
**当年横空出世，曾经惊艳全球的特斯拉，现在对消费者的吸引力迅速下降，特斯拉神话正在慢慢褪色。**

今年一季度，特斯拉共交付38.7万辆，和去年一季度同比下降了8.5%，这也是过去4年来特斯拉销量首次季度同比下降。如果和去年四季度环比，特斯拉的销量下降幅度更是超过了20%。

对于销量大幅下滑，特斯拉的解释是红海冲突导致物流受阻、德国工厂纵火案引发停工，美国加州工厂还处于产能爬坡阶段等等。不过，这些只是影响特斯拉销量很轻微的因素，最主要的原因还在于，
**全球电动汽车行业的竞争已经到了白热化阶段，而作为这个行业的领跑者，特斯拉曾经难以撼动的竞争优势正在快速下降。**

《无所畏惧》剧照

在特斯拉的全球销量版图中，中美是最重要的两大市场，占比接近7成，但是， **特斯拉在中美两大市场都正在面临前所未有的巨大挑战。**

最近几年，中国的电动汽车强势崛起，中国去年更是历史上首次成为全球最大的汽车出口国。

在中国诸多的电动车制造商中，尤以比亚迪势头最为凶猛。凭借电池技术上的巨大优势，加上极其丰富的产品线，比亚迪的销量爆发式增长。2021年，比亚迪汽车销量为70多万，去年就已经超过了300万辆，其中超过一半为纯电动汽车。尤其是在去年最后一个季度，比亚迪的纯电动汽车销量首次超越了特斯拉，成为全球销量最高的电动车制造商。

 **中国新能源车的崛起，对特斯拉带来了巨大压力。** 在中国新能源车的围堵之下，特斯拉从2022年开始掀起价格大战，Model
3的最低价格一度降到23万元左右。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/RNpp5IDIhiaJiclWJZcxUdZkmibYsWoiaV0vnBacScXLIV6kanoPaNbDnGCvtia3Viaa2Rc7hz1CPGMkianiaiarP9QEfZw/640?wx_fmt=jpeg&from;=appmsg)

《新闻女王》剧照

特斯拉的价格大战对“蔚小理”等造车新势力带来了很大的压力，但作为特斯拉最强大的对手，比亚迪却毫不畏惧。由于比亚迪在电池技术和生产成本上具有巨大优势，加上产能爆发式增长，规模效应也带来生产成本下降，比亚迪在今年年初更是主动降价出击，喊出了“电比油低”的口号。今年3月份，比亚迪的销量同比增长超过46%，环比增长更是高达148%。

相比之下，过去两年一直主动掀起价格战的特斯拉，现在的价格战略反而开始摇摆，时而降价，时而又涨价。既想降价提升销量，同时还要顾虑利润空间。

 **在美国，特斯拉面临的是另外一层压力，为了保护美国传统汽车制造商的利益，美国政府对于电车的扶持力度开始下降。**

今年3月下旬，美国政府放宽了汽车尾气排放标准，同时下调了未来几年的电动车发展目标。美国以前计划在2032年将纯电动汽车占新车的比例达到67%，但是最近将这一目标大幅下调到35%。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/RNpp5IDIhiaJiclWJZcxUdZkmibYsWoiaV0ve1EV9nWLVaGiaSMoQ5qIObSebaOLRRX8DFLtoAtX84Dq2JWCb0k64fA/640?wx_fmt=jpeg&from;=appmsg)

《黑暗荣耀》剧照

 **美国政府下降电动车目标，也就意味着对电动车相关配套产业的投入将大大减少，比如充电桩的建设等等，这将在很大程度上减少消费者购买电动车的动力。**

在美国，由于电车的竞争压力不如中国，所以，以特斯拉为首的电车价格还比较贵，加上充电桩数量不足，而美国人对于汽车驾驶里程的要求更高，所以，美国消费者对于电车的兴趣开始下降。去年年底，美国新车市场的纯电车销量占比大概为8%，但是今年2月份下降到只有6%。

 **很多美国消费者开始放弃纯电车，对混动汽车的兴趣明显上升。** 去年美国油电混动汽车的销量大幅增长了近80%，在美国市场的占比超过了纯电动汽车。

对于特斯拉而言，中国和美国是最大的两个市场，现在分别面临巨大挑战，中国市场的竞争越来越激烈，而美国开始降低对电动车的扶持，消费者对电车的兴趣也明显下降。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/RNpp5IDIhiaJiclWJZcxUdZkmibYsWoiaV0vvd01UeY4ea6ibicvOc0nJIYqlqqSZfpfiao2pJGlcwfXp9V8PCqlfDzyg/640?wx_fmt=jpeg&from;=appmsg)

《以爱为营》剧照

除了市场大环境在发生重大变化，特斯拉自身产品型号单一和老化，也在很大程度上降低了对消费者的吸引力。目前特斯拉的销量主力车型只有两款，Model
3和Model Y，这两款车型分别在2016年3月份和2019年3月份发布，距离现在已经长达8年和5年时间。

在这段时间，电动汽车的发展已经日新月异，从竞争对手的最新产品来看，无论是硬件还是软件都有了大幅升级。特斯拉的Model 3和Model
Y虽然依然算得上经典，但在很多消费者看来，也多少有些过时老化和审美疲劳。 **特斯拉推出新产品的速度过慢，也在很大程度上导致了消费者流失。**

对此，马斯克也在今年年初承认，特斯拉现在正处在两波增长浪潮的中间，第一波增长浪潮由Model 3和Model
Y带动，而下一波浪潮预计由更便宜的下一代电车推动。也就是市场所传言的廉价版本Model 2。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/RNpp5IDIhiaJiclWJZcxUdZkmibYsWoiaV0vbjvReRiaFJMAm8UF9LDakzVhPdDSEdoeO2u9KGHvbdEt8Muohy95pxw/640?wx_fmt=jpeg&from;=appmsg)

《都挺好》剧照

不过，在廉价版面世之前，特斯拉不得不长期面对竞争对手蚕食、销量继续下滑的现实。廉价版的特斯拉何时能够推出，俨然已经成了拯救特斯拉的救命稻草。最近有市场传言表示，特斯拉可能放弃了生产廉价版的计划，不过马斯克对此予以公开辟谣。

不过，即使特斯拉坚持推出廉价产品，在更便宜的价格区间，中国电动汽车已经卷成了一片红海，
**当年以高端形象面世的特斯拉，如果将来想凭借廉价产品和中国电车血拼，又能有几分胜算？**
