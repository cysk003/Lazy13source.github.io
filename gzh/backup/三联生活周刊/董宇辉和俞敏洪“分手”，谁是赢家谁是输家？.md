# 董宇辉和俞敏洪“分手”，谁是赢家谁是输家？

[**返回列表页**](/gzh/三联生活周刊)

***本文为「三联生活周刊」原创内容**

文｜谢九纠缠已久的董宇辉和东方甄选事件终于画上句号。7月25日，东方甄选发布公告，正式宣布和董宇辉分手，董宇辉以7658万元收购“与辉同行”。俞敏洪随后发布公开信，详细解释了双方“分手”的前因后果。
**按照协议，俞敏洪对于和董宇辉的“分手安排”可谓极其大度，不仅放手让董宇辉单飞，而且还给足了资金和资源扶持。**
俞敏洪在公开信中表示，“从今天开始，东方甄选的全资子公司与辉同行，将正式从东方甄选独立出去，由董宇辉个人持有其100%股权。”俞敏洪表示，“宇辉买公司的钱我安排了，公司是送给宇辉的。”俞敏洪还表示，“为了向宇辉表示感谢，除了已经支付承诺的全部待遇，我已恳请董事会及薪酬委员会并取得同意，把与辉同行的全部净利润奖励给宇辉。”公开资料显示，与辉同行公司2023年12月成立，截止今年上半年实现净利润1.41亿元。为了确保与辉同行业务的正常运营，东方甄选还将研发好的信息系统无偿交付与辉同行使用。和前几年李子柒事件相比，这一次董宇辉分手事件，俞敏洪表现出来的大度超出了很多人的预期。
**那么，俞敏洪为何如此呢？**![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/XnMeqb0xcz4Ak8p0jX2QdGfhYXhDVQBYicXeAT68YL6iaaymG5n8yvDo6HXDFicPfLs1zzArzeDBDsYQwu2AvFThg/640?wx_fmt=jpeg&from;=appmsg)俞敏洪抖音评论补充声明
**除了公司经营层面的考虑外，很大程度还和俞敏洪维护个人形象有关。**
前几年国内整顿教育平台，新东方几乎面临一夜清零的毁灭式打击。对于已经财务自由的俞敏洪，原本可以就此躺平，不过，出于对公司几万名员工的责任感，60岁的俞敏洪走进直播间，带领新东方再次转型。在俞敏洪的坚持和董宇辉等主播的共同努力下，新东方再度重生，股价从底部暴涨10几倍，创出历史新高。
**经此一役，俞敏洪收获了前所未有的好评，个人形象远远超过了他此前的新东方时期。**

![](https://mmbiz.qpic.cn/mmbiz_png/c2Sib3Mp7pOONCIAopiblTibnxLRtcFQ8bvEVibjumEFUnRF2RNVDPYFhaziap8s3rO0Dsqdvy256kySjSR5WOkmfng/640?wx_fmt=other&from;=appmsg&tp;=webp&wxfrom;=5&wx;_lazy=1&wx;_co=1)

俞敏洪和董宇辉同框直播间做“小作文”事件总结不过，随着董宇辉的光芒越来越耀眼，大主播和平台之间的矛盾开始掩盖不住浮出水面。2023年12月，东方甄选爆发小作文事件，网友怀疑董宇辉在东方甄选被排挤打压，俞敏洪作为当家人，成为很多网友指责的对象。按照俞敏洪最近在股东沟通会上的描述，“近三个月，我的形象因网暴受到巨大损失，网络上关于我的阴谋、攻击、虐待员工等言论络绎不绝，好像我成了全世界最凶恶的资本家和老板。”
****网民对于俞敏洪的攻击，不仅让俞敏洪的个人形象受损，同时也影响到了东方甄选的业务运营。小作文事件之后，虽然俞敏洪免去了原CEO孙东旭的职务，力挺董宇辉。
**不过，网民对于俞敏洪的攻击并没有停止。**

![](https://mmbiz.qpic.cn/mmbiz_png/c2Sib3Mp7pOOjq2kGFJHomb3wzbOJlEM4gaGo6sNN9ib4009FLlicbc4HbxDbWwBgxFSY5bOEDAjWDkazVIs4JVTA/640?wx_fmt=png&from;=appmsg)俞敏洪相关视频截图

俞敏洪在公开信中表示，“我在与辉同行镜头面前出现减少了，主要原因是因为我一旦出现，就会出现密集的攻击；我也不敢出现在东方甄选的镜头面前，一旦出现，也是密集的攻击。我左右为难，作为东方甄选的创始人，支持公司发展变成了一件极其困难的事情。”
**“小作文事件”之后，即使董宇辉继续留任，但东方甄选的股价还是从30港元附近一路下跌到12港元左右，这也让俞敏洪意识到必须做出改变，否则东方甄选没有出路。**
**正是在这样的背景下，俞敏洪选择了放手，让董宇辉离开东方甄选单飞。**
而且，为了回应网民之前的“铁公鸡”言论，俞敏洪在分手安排上表现得极为大度。俞敏洪希望以此挽回过去几个月以来的个人形象损失。对于有些已经财富自由的人而言，个人形象和外界评价，比财富更加重要。
**不过，俞敏洪在尽力向网友挽回个人形象的同时，却忽略了东方甄选股东们的感受。**
在和董宇辉的分手协议公布之后，7月26日，东方甄选的股价暴跌24%，市值一天之内蒸发了20亿港元，对于东方甄选的股东们而言，这是最真实的切肤之痛。![](https://mmbiz.qpic.cn/mmbiz_png/c2Sib3Mp7pOOjq2kGFJHomb3wzbOJlEM4gEFGGnlnT9KbUsWHqmkqPNGwCZnZ8jt1pKfzlEmS8Y1tOr5xvCOmYA/640?wx_fmt=png&from;=appmsg)东方甄选直播间俞敏洪表示，“出于信息保密等原因，我没有办法和大家进行实时沟通和交流，导致了一些股东对我的失望和愤怒，我后续一定会以加倍的努力，把更多的精力投入到东方甄选的运营管理中，来报答大家，努力给大家一个发展的越来越好的东方甄选。”不过，在失去了董宇辉之后，东方甄选是否还能回到曾经的高度，无疑具有巨大的不确定性，公司股价暴跌24%，除了表达股东的愤怒情绪之外，更是投资者对于东方甄选未来的最直接反应。
**如果俞敏洪早知道公司股价会如此暴跌，不知道会不会做出不一样的安排？**
**对于董宇辉而言，终于实现了单飞的梦想，表面上来看是个人的巨大胜利，但是未来如何也同样具有巨大的不确定性。**
主播带货这种模式在爆红几年之后，现在已经开始出现降温势头。从去年双十一开始到今年618大促，很多头部主播的带货量明显下滑，在头部主播带货模式萎缩的同时，各大品牌的自有直播间开始成为新的风口，越来越受到消费者的追捧。
**对于单飞之后的董宇辉而言，最大的挑战不只是行业趋势的转变，还有董宇辉自身的定位和发展。**![](https://mmbiz.qpic.cn/sz_mmbiz_gif/XnMeqb0xcz4Ak8p0jX2QdGfhYXhDVQBYQMoyIcibLibUI6Zgsw1T6zwpy1icg4XtsOrqyfAy9pVH51Oias81jib3nXw/640?wx_fmt=gif&from;=appmsg)《我爱我，很棒》剧照董宇辉的悖论在于，被网民贴上了文化人的标签，但是，一个真正有深度的文化人，内心深处应该不会满足于长期扮演直播卖货的角色。就连李佳琦都已经控制不住对直播带货的厌倦，在直播间里对网友公开表示，“有的时候找找自己原因，这么多年了工资涨没涨，有没有认真工作？
**对于文化人董宇辉而言，显然也就更加不甘长期做一个直播带货人。**
今年以来，董宇辉就多次公开表示对直播带货的抵触心理，表示自己非常抗拒卖东西，到今天都不享受直播带货的工作。每天需要吃安眠药来帮助睡眠，对于网红这个称呼也感到反感。
**董宇辉认为“我自诩读书人，世界不缺我一个卖货的，成为带货主播和销售冠军并不是我的本心”**![](https://mmbiz.qpic.cn/sz_mmbiz_gif/XnMeqb0xcz4Ak8p0jX2QdGfhYXhDVQBYTDNRYLmsiaNfwqViaDtDn910pd6BaAzofMXrYagjZmMV8LuqqBKN2d3Q/640?wx_fmt=gif&from;=appmsg)图源央视网采访这一次从东方甄选单飞之后，董宇辉成为自己的老板，他需要将更多精力用于扮演管理者的角色，对于董宇辉而言，这是一个全新的挑战。
**而对于将他捧红的粉丝们而言，更希望看到的是一个在直播间里妙语连珠的董宇辉，而不是一个隐藏在幕后的管理者。**
