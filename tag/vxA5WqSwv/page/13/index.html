(提问)枫大少 : 请教花爷，县城的土地有没有投资价值？在老家县城，城边有大量工业用地，未来规划要扩建成公园，目前价格大概6万/亩，前年的时候2万/亩，不知道可否有投资价值？2019-10-29回答：回答不了。 说下，星球里基本涉及到职场、副业、掘金、生意靠不靠谱、甚至情感（因为情感其实也是利益关系）的问题，都会回答。 但是花爷不回答投资相关的具体问题。 比如我自己会买股票，但你问我，股票A和股票B花爷你买了哪个？抱歉，不能答。 你问我，马上要结婚了现在买房是价格低谷吗？我可以告诉你，刚需现在买没啥问题。但如果你问我具体到XX楼盘值不值得买？抱歉，不能答。 究其原因，是因为投资是件操作起来很简单（投钱就行），但结果可能影响特别大的事情。投资金额越大，涉及到的分析背景越复杂，比如县城土地投资这个问题，要确定值不值得投，花一个星期做详细调查也不过分。就题干这点信息，根本判断不出来。 再说了，都详细了投资的具体项目了，我要能十拿九稳的预言成功，我该收多少钱啊？ 所以投资趋势上的问题，能聊两句。但投资相关的具体问题，干脆，直接不回答，对你们负责，我也省心，见谅。点赞数:9评论区：马头的班长 : 我很好奇，土地属于一级市场，目前个人是没法买卖土地的，你是开发商？ 花爷梦呓换酒钱 : 企业可以拍啊，土地又不是只能盖房子

金花奖挑战现在

2019-10-28 #愚公掘金
(提问)Nora : #金花奖挑战#现在不少在线教育公司，通过城市合伙人/推广人等方式，利用在当地有资源的人或机构来下沉三四线城市推广。这类合伙人的招募不像其他的奶茶加盟等等大肆的进行信息流投放，有部分会做sem投放或者买招商加盟网的资源位。行业里，像樊登读书会靠个人ip，城市合伙人推广的很好。请花爷分析下，城市合伙人有没有比较好的推广/招募方式？感谢2019-10-28回答：我们经常讲to B和to C，有花友问到底啥是B啥是C啊？其实B就是企业，C就是个人用户。 传统招商是典型的to B行业，而to B的特点，就是在特定的用户池里捞人。因为B端的企业用户相对来说比较少，而且分散，你只有到目标的B端用户聚集的地方，获客才有效。 所以传统招商怎么搞？买加盟网的广告、买招商加盟的搜索关键词广告、线下开行业大会做会销，都是把目标用户聚起来再捞人的打法。 但现在时代变了。 以前说到B端用户，都是企业、公司、地方小老板，但现在有很多B端用户，本身并不属于B端，他们是从C端用户升级来的。 比如你说的樊登读书会，他们先是以读书为主题，获取了海量的C端用户，然后再在用户里找加盟商，因为有充分的信任关系，再加上加盟费用不高，一部分有经济实力的C端用户，就转化成了小B用户。 以前，是在池子里捞鱼，现在，是不管什么鱼，统统围起来，养一段时间，然后挑大的捞。 这两种合伙人招募方法，哪个更有效？ 如果你要找少量的合伙人，老式的捞人方法更有效，短时间就能完成任务，但数量上很难突破天花板。 如果你要找大量的合伙人，新式的围养方法更有效，产品和产品在社交网络的传播，是你最好的宣传。你会花一段时间先围养用户，但一旦围养成功，合伙人数量只与用户总数成正比，几乎没有天花板。点赞数:12评论区：Nora : 感谢花爷的回复。您所述的两种方案，第一种从C端用户中转化是我们前期已经在尝试的模式，初期有一些转化。目前想同步尝试从“B端”的效果。但花爷将目标群体进行toB和toC区分的思考方式，对我有不少启发。再次感谢[爱心] 花爷梦呓换酒钱 : b端就是不停是试渠道了，少量多次，找到杠杆高的渠道 花爷梦呓换酒钱 : 金花奖挑战成功！加我微信：huayebenren，发送星球内容截图领取红包奖励。大家如果觉得这条内容不错，记得点赞，本周点赞最多的金花奖内容，会有额外的红包翻倍奖励。
