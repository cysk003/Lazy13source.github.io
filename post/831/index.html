评论区：
华 : 请问是汉堡店赚还是火锅店赚呢，两者的优势分别是什么呢
😊 : 我这两个店都赚钱，不过火锅店相对赚多一点。火锅店相对于汉堡店优势在于消费品类的池子大。现在火锅是除了家常菜的第一体量的消费市场。消费群体受众比较广，而是消费客单价比较高。缺点是北方淡旺季比较明显。汉堡店优势在于投资相对较小，程序简单好复制，相比于其他小吃利润率还可以，不过现在快餐的发展汉堡店的红利基本吃完了。
碧海潮生 : 德克士还是华莱士，15块钱3个汉堡的那种，第一次见把价格战打的这么狠的[呲牙][呲牙]
😊 : 汉堡店现在已经没什么可以升级的了，肯德基推出那么多新品尝鲜的依然是香辣鸡腿堡。头部巨头都创新无力，现在也在转型改变模式。底下的小品牌没有什么研发能力和品牌知名度只能打价格战了。
CC : 近期有开餐厅的打算，选址有什么好的方案，头疼中
😊 : 选址要看你开的餐厅的定位是属性是什么，是社区还是流量型的。之前没有餐厅经验别轻易进来。
